# Corpus Usage Guide

This document explains how to read, interpret, and use the
**Ethical Narrative Corpus** as a long-form narrative dataset.

This corpus is not designed for casual browsing.
It is designed for **sequential exposure and contextual accumulation**.

---

## 1. Intended Reading Mode

The Ethical Narrative Corpus is meant to be read **in sequence**.

Each narrative unit assumes:
- prior context
- accumulated relationships
- remembered consequences
- unresolved tensions from earlier segments

Random sampling, shuffling, or isolated extraction
will significantly degrade the meaning and alignment signal
encoded in the corpus.

This is not a flaw.
It is a deliberate design choice.

---

## 2. Why Sequential Context Matters

Human ethical judgment does not emerge from isolated dilemmas.
It emerges from:

- memory of past actions
- responsibility carried over time
- repeated interaction with the same individuals
- long-term cost of short-term optimization

The narratives in this corpus are structured to reflect this reality.

Removing temporal continuity reduces the corpus to surface-level text.
Preserving continuity turns it into an **ethical environment**.
